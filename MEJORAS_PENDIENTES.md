# 📝 Próximas Mejoras para TacoManagment

Este documento contiene las ideas y características planificadas para el bot, divididas por categorías.

## 🏅 Sistema de Reputación y Gamificación
*   **Comando `/profile`**: Mostrar tarjeta visual con nivel, sugerencias aceptadas y tickets abiertos.
*   **Roles por Actividad**: Asignación automática de roles (ej. "Sugeridor Experto") tras X sugerencias aceptadas.

## 📅 Sistema de Anuncios Programados
*   **Comando `/announce schedule`**: Programar mensajes únicos (eventos, reinicios).
*   **Mensajes Recurrentes**: Configurar mensajes que se reenvían cada X tiempo para mantener visibilidad.

## 🤖 Verificación Anti-Raid Avanzada
*   **Verificación por Botón**: Sistema simple para nuevos usuarios que desbloquea el servidor.
*   **Logs de Sospecha**: Alertar al staff sobre cuentas creadas recientemente (<24h).
